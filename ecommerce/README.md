## ecommerce
珠寶電商網站


### Features
- 使用者可在網站內購買珠寶，包括產品下訂、結帳、付款。
- 網站管理員可使用後台管理功能新增產品以及編輯訂單。


### prerequisites
- vue: ^2.5.2
- axios: ^0.21.1
- jquery: ^3.6.0


### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
